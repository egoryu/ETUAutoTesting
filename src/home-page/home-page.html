Описание использованных тэгов

На этой странице используются следующие HTML5 тэги:

  • <table> - для отображения таблицы соответствия букв и номеров вариантов
  • <article> - для семантического выделения описательной части
  • <input> - для поля поиска вариантов
  • <button> - для навигационной кнопки
