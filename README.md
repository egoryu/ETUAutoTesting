# Инструкция по запуску проекта

1.  Скачайте [Node.js](https://nodejs.org/) с официального сайта.
2.  Скачайте [Git](https://git-scm.com/) с официального сайта.
3.  Установите Angular CLI глобально:  
    `npm install -g @angular/cli@latest`
4.  Клонируйте репозиторий:  
    `git clone https://github.com/egoryu/ETUAutoTesting.git`
5.  Перейдите в папку проекта:  
    `cd ETUAutoTesting`
6.  Установите зависимости:  
    `npm install`
7.  Запустите сервер разработки:  
    `ng serve`
